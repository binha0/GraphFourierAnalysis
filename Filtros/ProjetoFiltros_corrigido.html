Nesse projeto, visamos comparar diferentes tipos de filtros com relao a diferentes rudos em uma mesma imagem.

Os filtros propostos so o filtro do kernel da equao do calor e tambm o de regularizao Tikhonov, este sendo um caso particular de um filtro $ARMA(p,q)$.

Eles so definidos por:

$$\text{heat} (\lambda) = e^{- \tau \lambda}$$

onde $\tau$ um parmetro real a ser definido, e

$$\text{Tikhonov} (\lambda) = \frac{1}{1 + \gamma \lambda}$$

onde $\gamma$ um parmetro real a ser escolhido.

Ambos os filtros tem implementao na biblioteca PyGSP, com a qual faremos a comparao de desempenho no final do trabalho.

Abaixo, esto definidas as funes utilizadas para calcular os filtros Heat e Tikhonov, bem como as mtricas de Erro Quadrtico Mdio (MSE) e Peak Signal-to-Noise Ratio (PSNR).

Na ordem:

  • Heat
  • Tikhonov
  • MSE
  • PSNR
In[]:
def heat(lam,tau=1):
    '''Calcula a funo do filtro a partir de um vetor de autovalores.
    
    Parmetros:
    - lambda (array) - vetor de autovalores
    
    Return:
    - H (array) - matriz diagonal cujos calores da diagonal so os h(lambda_i) e
    h()  a funo que define o filtro
    '''  
    
    h_lambda = np.exp(-(tau * lam)) #h(lam,tau)
    H = np.diag(h_lambda)

    return H
In[]:
def tikhonov(lam,g=1):
    '''Calcula a matriz H da regularizao  tikhonov a partir de
    um vetor de autovalores.
    
    Parmetros:
    - lam (array): vetor de autovalores
    - g (float): default = 1, valor do parmetro lambda 
    
    Returns:
    - H (array): matriz diagonal cujas entradas so a funo h() 
    calculada sobre os autovalores correspondentes.'''
    
    h_lambda = 1/(1+ g*lam)
    H = np.diag(h_lambda)

    return H

def MSE(original,filtrado):
    '''Calcula o erro quadrtico mdio da filtrao escolhida.
    
    Parmetros:
    - original (array): sinal original, sem rudos, para usar como
    referncia no MSE
    - filtrado (array): sinal obtido com a filtrao
    
    Returns:
    - mse (float): erro quadrtico mdio'''

    soma = sum((original - filtrado)**2) # soma dos quadrados 
    n = len(filtrado)
    mse = soma/n
    return mse
In[]:
def PSNR(original,filtrado,img=True):
    '''Calcula o Peak signal-to-noise ratio com relao a alguma fltrao
    escolhida. O PSNR  a variao relativa do mximo valor possvel
    do sinal em relao ao erro quadrtico mdio (norma L2 do erro),
    em escala logartmica.
    
    Parmetros:
    - original (array): sinal original
    - filtrado (array): sinal filtrado de acordo com a filtrao de 
    escolha
    
    Returns: 
    - res (float): medida do PSNR'''
    mse = MSE(original,filtrado) # precisa dessa mtrica
    if img:
        max_f = 255
    else: 
        # definindo uma mtrica alternativa para meu toy problem
        max_f = max(filtrado)
    res = 20*np.log10(max_f) - 10*np.log10(mse)
    return res

# primeiro a identificao das bordas
def bordas(img):
    '''Funo que recupera os ndices de cada uma das bordas de um array (ou imagem).
    
    Parmetros:
    - img (array): matriz mxn que representa a imagem (ou qualquer outra coisa)
    
    Return:
    - left (list): ndices da borda esquerda 
    - right (list): ndices da borda direita
    - up (list): ndices da borda superior
    - down (list): ndices da borda inferior'''
    # matriz m por n 
    m, n = img.shape 

    left = [i*m for i in range(n)]
    right = [i*m - 1 for i in range(1,n+1)]
    up = [i for i in range(m)]
    down = [n*(m-1) + i for i in range(m)]

    return left, right, up, down

def grafo_img_simples(img, adj = False, plot=True):
    '''Funo que retorna uma instncia do objeto graph do pygsp que
    representa uma imagem. Aqui o grafo  o mais simples possvel, considerando
    adjascentes os pixels vizinhos.
       
    Parmetros:
     - img (array): imagem a partir da qual construiremos o grafo 
     - adj (Bool): default = False, indica se retornamos ou no a matriz de adjascncia
      do grafo criado 
     - plot (Bool): default = True, faz o desenho do grafo
    '''
    # primeira coisa  pegar as bordas 
    left, right, up, down = bordas(img)
    m,n = img.shape
    #W = np.zeros((m*n,m*n))
    W = []

    for i in range(m*n):
        linha = np.zeros(m*n,dtype=np.int16)

        if i == 0: # canto superior esquerdo 
            linha[1] = 1 # a direita
            linha[m] = 1 # abaixo
            linha[m+1] = 1 # diagonal p baixo

        elif i == m-1: # canto superior direito
            linha[i-1] = 1 # a esquerda 
            linha[i+m] = 1 # abaixo 
            linha[i+m-1] = 1 # diagonal p baixo

        elif i == n*(m-1): # canto inferior esquerdo 
            linha[i+1] = 1 # a direita
            linha[i-m] = 1 # acima
            linha[i-m+1] = 1

        elif i == n*m -1: # canto inferior direito
            linha[i-1] = 1 # a esquerda
            linha[i-m] = 1 # acima
            linha[i-m-1] = 1

        elif i in up:
           linha[i-1] = 1
           linha[i+1] = 1
           linha[i+m] = 1
           linha[i+m-1] = 1
           linha[i+m+1] = 1
            
        elif i in left:
            linha[i+1] = 1
            linha[i-m] = 1
            linha[i+m] = 1
            linha[i-m+1] = 1
            linha[i+m+1] = 1
        
        elif i in right:
            linha[i-m] = 1
            linha[i+m] = 1
            linha[i-1] = 1
            linha[i-m-1] =1
            linha[i+m-1] = 1

        elif i in down:
            linha[i-1] - 1
            linha[i+1] = 1
            linha[i-m] = 1
            linha[i-m+1] = 1
            linha[i-m-1] = 1

        else:
            linha[i-m-1] = 1
            linha[i-m] = 1
            linha[i-m+1] = 1
            linha[i-1] = 1
            linha[i+1] = 1
            linha[i+m-1] = 1
            linha[i+m] = 1
            linha[i+m+1] = 1
        
        W.append(linha)
    W = sp.csc_matrix(np.array(W, dtype=np.int16))
    #assert W.shape == (m*n,m*n)
    
    # criando o grafo 
    G = graphs.Graph(W)
    
    if adj:
        return G, W
    elif plot:
        G.set_coordinates()
        G.plot(title=f"Grafo trivial para imagem {m}x{n}")
        return G
    else:
        return G

Grafo com Pesos

Nessa seo, introduzimos um novo mtodo para construo dos grafos, em que aplicado um peso Gaussiano quando dois vrtices so $\kappa$-prximos, ou seja,

$$a_{ij} = e^{- d^2 / 2 \sigma^2}, \text{se} |d(i,j)| < \kappa$$

A distncia aqui utilizada a diferena entre as cores de cada pixel em grayscale.

In[]:
def grafo_peso_grayscale(img,th,kap,adj=False,plot=True):
    '''Funo que constri o grafo para uma imagem em escala de cinzas
    a partir do gradiente de cor. Os pesos atribuidos a cada vrtice sero
    uma gaussiana de varincia theta^2 avaliada em em x = gray(i) - gray(j), 
     onde gray()  o valor na escala de cinzas de cada pixel.
      
    Parmetros:
     - img (array): matrix representando a imagem, suas entradas correspondem ao valor
     de cada vrtice na escala de cinza
     - th (float): desvio padro da Gaussiana
     - kap (float): threshold de conexo dos vrtices'''
    
    m,n = img.shape
    img_array = np.array(img).reshape(n*m)
    N = m*n
    W = np.zeros((N,N))
    #W = sp.coo_matrix(shape=(N,N),dtype=np.float16)
    for i in range(N):
        for j in range(N):
            dist = abs(img_array[i] - img_array[j])
            #print(img[ij,ii],img[jj,ji],dist)
            if dist < kap and i!=j:
                W[i,j] = np.exp(-((dist*dist)/(2*th*th)), dtype=np.float32)
                #print(W[i,j])
                assert W[i,j] <= 1 
    
    G = graphs.Graph(W)
    if adj:
        return G, W
    elif plot:
        G.set_coordinates()
        G.plot(title=f"Grafo com pesos $\kappa =$ {kap}")
        return G
    else:
        return G
